# satellite-predictor

Local satellite pass predictor for **Chagrin Falls, OH** — no ads, no browser, no rate limits.

Fetches TLE data from [Celestrak](https://celestrak.org) and uses SGP4 propagation (via `skyfield`) to calculate upcoming passes for NOAA weather satellites and the ISS.

## Satellites tracked

| Name    | NORAD  | Frequency     | Use                  |
|---------|--------|---------------|----------------------|
| NOAA 15 | 25338  | 137.620 MHz   | APT weather imagery  |
| NOAA 18 | 28654  | 137.9125 MHz  | APT weather imagery  |
| NOAA 19 | 33591  | 137.100 MHz   | APT weather imagery  |
| ISS     | 25544  | 145.800 MHz   | SSTV / amateur radio |

## Install

```bash
pip install -r requirements.txt
```

Dependencies: `skyfield`, `requests`, `rich`

## Usage

```bash
# Pretty terminal output — next 3 days, passes ≥ 20°
python3 predict.py

# More days, higher threshold
python3 predict.py --days 5 --min-elevation 30

# Single satellite
python3 predict.py --satellite "NOAA 19"

# JSON output (for scripting / crontab)
python3 predict.py --json > /tmp/passes.json

# Force TLE refresh from Celestrak
python3 predict.py --refresh

# Schedule next best pass in crontab (requires rtl_fm + sox)
python3 predict.py --schedule --min-elevation 30
```

## Crontab scheduling (`--schedule`)

`--schedule` finds the highest-elevation upcoming pass and writes two crontab entries:

```cron
# AUTO-GENERATED by predict.py — do not edit manually
59 22 9 5 * /Users/billfordx/record_noaa.sh 137.100M
31 23 9 5 * pkill -f rtl_fm
# END AUTO-GENERATED
```

The block is replaced on every run — safe to call repeatedly.

### Daily hands-off operation

Add this to your crontab to refresh TLEs and reschedule every morning at 08:00:

```cron
0 8 * * * /usr/local/bin/python3 /Users/billfordx/predict.py --schedule --min-elevation 30
```

## Recording script (`record_noaa.sh`)

Copy to `~/record_noaa.sh` and `chmod +x`:

```bash
./record_noaa.sh            # NOAA 15 (default)
./record_noaa.sh 137.100    # NOAA 19
./record_noaa.sh 145.800    # ISS
```

Requires `rtl_fm` and `sox` installed via Homebrew.

## TLE cache

TLE files are cached in `./tle_cache/` and refreshed automatically when older than 24 hours. Use `--refresh` to force an immediate update.

## Configuration

Edit the constants at the top of `predict.py`:

| Variable            | Default          | Description                      |
|---------------------|------------------|----------------------------------|
| `OBSERVER_LAT`      | `41.4270`        | Observer latitude                |
| `OBSERVER_LON`      | `-81.3990`       | Observer longitude               |
| `OBSERVER_ALT`      | `300`            | Observer altitude (meters)       |
| `DAYS_AHEAD`        | `3`              | Default prediction window (days) |
| `MIN_ELEVATION`     | `20`             | Default minimum elevation (°)    |
| `TLE_MAX_AGE_HOURS` | `24`             | TLE cache TTL                    |
| `RECORD_SCRIPT`     | (path)           | Path to `record_noaa.sh`         |
